h1. ROBO-0023 Arm3 

_An open source palletizing robot based on the ABB 460_

This is the Arduino firmware that lives in the arduino brain of the robot.  It empowers the robot to understand inverse kinematics and move according to gcode commands that you give it.

This project is designed to pair the human-friendly interface "Evil Overlord":https://github.com/i-make-robots/Evil-Overlord
